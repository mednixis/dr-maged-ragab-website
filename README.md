# drmagedragab.com

Production website package for Dr. Maged Ragab.

## What This Contains

- Public website pages
- Admin login prototype
- Backend/CRM dashboard prototype
- Database structure page
- Production launch checklist
- Supabase database schema
- Supabase RLS/security policies
- Payment integration plan
- DNS setup notes

## Main Links After Deployment

- Public website: `/index.html`
- Booking: `/booking.html`
- Admin login: `/admin-login.html`
- Admin / CRM dashboard: `/admin.html`
- Database structure: `/database.html`
- Production launch plan: `/launch.html`

## Deployment

Upload the contents of this folder to the root of the GitHub repository, then import the repository into Vercel.

Recommended domain setup:

- `drmagedragab.com`
- `www.drmagedragab.com`
- `admin.drmagedragab.com`
- `api.drmagedragab.com`

## Backend Setup Files

Run in Supabase SQL Editor:

1. `database-schema.sql`
2. `supabase-rls-policies.sql`

Use `env.example` as the production environment variable checklist.
