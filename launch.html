Download Build Orderتحميل ترتيب التنفيذ
01Buy Domainشراء الدومين

Buy drmagedragab.com using the owner or Mednixis email. Keep auto-renew on and verify the registrant email.شراء drmagedragab.com ببريد المالك أو Mednixis. تفعيل التجديد التلقائي وتأكيد بريد المالك.

02Create Owner Accessإنشاء وصول المالك

Create owner@drmagedragab.com or admin@mednixis.com and use it for domain, hosting, database, payment, and backups.إنشاء owner@drmagedragab.com أو admin@mednixis.com واستخدامه للدومين والاستضافة وقاعدة البيانات والدفع والنسخ الاحتياطي.

03Deploy Websiteنشر الموقع

Deploy public website at www.drmagedragab.com and admin dashboard at admin.drmagedragab.com.نشر الموقع على www.drmagedragab.com ولوحة الإدارة على admin.drmagedragab.com.

04Create Databaseإنشاء قاعدة البيانات

Create Supabase project, run database-schema.sql, then run supabase-rls-policies.sql.إنشاء مشروع Supabase وتشغيل database-schema.sql ثم supabase-rls-policies.sql.

05Connect Loginربط تسجيل الدخول

Connect admin login to Supabase Auth with owner, admin, secretary, doctor, and content editor roles.ربط دخول الإدارة مع Supabase Auth بصلاحيات المالك والمدير والسكرتارية والطبيب ومحرر المحتوى.

06Connect Paymentsربط الدفع

Use Paymob or PayTabs hosted payment page for international Zoom deposit only.استخدام صفحة دفع آمنة من Paymob أو PayTabs لإيداع Zoom الدولي فقط.

07Connect CRMربط CRM

Send booking confirmation location messages and end-of-day review messages through WhatsApp/SMS.إرسال رسائل تأكيد الموقع ورسائل تقييم نهاية اليوم عبر واتساب/SMS.

08Secure Everythingتأمين النظام

Enable HTTPS, 2FA, RLS, private report storage, webhook verification, backups, and audit logs.تفعيل HTTPS و2FA وRLS وتخزين التقارير الخاص والتحقق من Webhooks والنسخ الاحتياطي وسجل المراجعة.
